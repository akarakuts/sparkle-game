# Changelog

## 1.4.1 — 2026-06-10

### Исправления
- Прогресс мини-игр: завершение больше не отмечает неверный слот (например, «Рисование» вместо «Сортировки»).
- `SceneManager` сразу передаёт `world_id` / `game_id` в сцену мини-игры.
- Корректная обработка дублей типов мини-игр в мирах 4 и 5 (две «Последовательности» / две «Сортировки»).
- Регрессионные тесты для всех 7 миров.

## 1.4.0 — 2026-05-29

### Архитектура
- `GameConstants` — единый источник путей сцен и числа миров.
- `ProgressService` — логика завершения мини-игр вынесена из `SceneManager`.
- `MinigameBase` / `StrokeCanvas` — общие хелперы для мини-игр и рисования.

### Игровой процесс
- **ParentGate** — случайная арифметическая задача вместо фиксированного ответа.
- **Memory** — размер сетки зависит от сложности (3×2 … 4×4).
- **Drawing** — минимальное число штрихов зависит от сложности.
- **Puzzle** — удалён неиспользуемый `Area2D` для drop-зон.

### UI
- Цифровые текстуры централизованы в `MiniGameArt.DIGIT_TEXTURES`.
- Счётчик осколков на карте миров упрощён (дублирующий shard UI убран).

### Android
- `screenOrientation=portrait` в манифесте.
- Отключён `profileable` в release-сборке.

### CI / тесты
- Тесты (`smoke_test`, `test_runner`) перед release-сборкой.
- Кэш export templates в GitHub Actions.
- OSV: `fail-on-vuln: true`.
- Тесты миграции сейвов и идемпотентности `complete_world`.
